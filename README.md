# grunt-set-node

A set of grunt tasks to help with node library development.

## Features

* jshint js files
* watch files for changes and automatically run correct task
* run mocha tests

## Suggested Folder Structure

```
lib/
  lib.js
test/
	lib.test.js
	index.html
index.js
```

## Commands

* `dev` - start up dev server and watch for changes
